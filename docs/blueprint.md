# **App Name**: FinTrack Pro

## Core Features:

- Transaction Tracking: Record income and expenses with descriptions, categories, and timestamps.
- Dashboard Summary: Display key financial stats like balance, income, and expenses after a transaction is added, with transaction details accessible via click.
- Budgeting: Set monthly spending limits per category and track spending progress.
- Savings Goals: Define financial goals with target amounts and deadlines, track progress over time.
- AI-Powered Financial Assistant: Ask about current balance, income, expenses, budgets or goals, or request a financial tip. Implement a tool which may or may not choose to incorporate details like account balances or past spending depending on context.
- Automated Insights: Suggest potential categories when logging transactions, provide automated analysis and budget advice.  Make payments in real-time to see their outcome after saving any type of transactions
- User Authentication: Secure user accounts with login/signup functionality.

## Style Guidelines:

- Primary color: Deep indigo (#4F46E5) for a sense of trust and sophistication.
- Background color: Light gray (#F9FAFB) to ensure a clean and modern interface.
- Accent color: Teal (#2DD4BF) for highlighting interactive elements and key information.
- Body and headline font: 'Inter', a sans-serif, for a clean and neutral aesthetic.
- Use 'Lucide React' icons, maintaining a consistent stroke width and a simple, modern style.
- Employ a card-based layout with subtle 3D effects for enhanced visual appeal.
- Use smooth, subtle transitions and animations to provide feedback and enhance the user experience; ensure animations are 'simple and aestithcal'.
